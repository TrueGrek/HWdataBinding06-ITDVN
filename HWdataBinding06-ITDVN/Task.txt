Задание 2
Используя Visual Studio, создайте проект по шаблону ASP.NET Empty Web Application.
Создайте SQL базу данных с таблицей Contacts. Добавьте в таблицу следующие колонки: FirstName, Lastnamed, Phone, Email, Comments.
Создайте страницу для внесения данных в базу.
Создайте страницу, на которой данные из базы будут отображаться с помощью элемента управления
DetailsView.

Задание 3
Используя Visual Studio, создайте проект по шаблону ASP.NET Empty Web Application.
Добавьте в проект две страницы. Survey.aspx (Анкета). 
Страница Survey.aspx должна содержать форму с регистрационной анкетой. Разработайте структуру
базы данных, в которой будут сохраняться данные введеннные в анкетах.
Анкета состоит из следующих полей.
 Фамилия.
 Имя и Отчество.
 Название конкурсной работы.
 Номинация конкурсной работы (список для выбора):
o Учебный проект
o Воспитательный проект
o Поза школьный проект
o On-line образовательный ресурс
o Учебно-методическое обеспечение
 Дисциплина (например, биология, математика).
 Была ли апробирована Ваша анкета (Да, Нет)?
 Если да, то на каком уровне?
 Полное название организации в которой Вы работаете.
 Ваша должность.
 Город, населенный пункт.
 Область.
 Мобильный телефон.
 Электронная почта.
 Как Вы о нас узнали:
o Сайт МОН
o Сайт Майкрософт Украина
o Сайт сети «Партнерство в обучении»
o Областное управление образованием
o Городское управление образованием
o Osvita.com
o Gurt.org.ua
 Поле ввода для своего варианта «Как Вы о нас узнали»
В конце анкеты должен находится CheckBox с надписью «Я даю соглашение на обработку моих личных
данных и соглашаюсь с политикой конфиденциальности (ссылка на страницу с правилами)».
Если checkbox не отмечен, данные анкеты не сохраняются в базу, вместо этого отображается сообщение
об ошибке.
